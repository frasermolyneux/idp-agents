## Overview
Azure Functions-based agent platform for the Internal Developer Platform. Hosts Durable Task orchestrations, MCP tool endpoints, campaign automation, proactive ops, and RAG indexer.

**Tech Stack**: .NET 9, Azure Functions (Flex Consumption), Durable Task Extension, Microsoft Agent Framework, MCP, Terraform, GitHub Actions
